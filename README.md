# Cognifyz Machine Learning Internship

This repository contains the projects completed as part of the Cognifyz Technologies Machine Learning Internship.

## Tasks Completed

### Task 1: Predict Restaurant Ratings
- Built a machine learning regression model
- Predicted restaurant aggregate ratings
- Performed feature importance analysis

### Task 2: Restaurant Recommendation System
- Developed a content-based recommendation system
- Used cosine similarity for restaurant recommendations

### Task 4: Location-Based Analysis
- Performed geographical analysis of restaurants
- Visualized restaurant distribution and city-wise insights

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab
